`combinations`를 사용하면 쉽게 풀 수 있다.

```python
import sys
from itertools import combinations


def solution():
    N, M = map(int, sys.stdin.readline().split())
    sequence = [i for i in range(1, N+1)]
    combs = combinations(sequence, M)
    
    for comb in combs:
        print(*comb)


if __name__ == "__main__":
    solution()

```
