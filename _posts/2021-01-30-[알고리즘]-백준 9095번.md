`1, 2, 3`으로 주어진 N을 만드는 모든 조합은 N에 도달 할때까지 `x+1, x+2, x+3`을 지속적으로 하면된다.

```python
x = 0
x의 경우의 수: [x+1], [x+2], [x+3] # 1st
x = 1, 2, 3
x의 경우의 수: [1+1], [1+2], [1+3] | [2+1], [2+2], [2+3] | [3+1], [3+2], [3+3] # 2nd
....
x == N 이 될 때까지 반복
```

위 아이디어를 구글링을 통해 알게 되었고 재귀적으로 구현했다.

```python
import sys

counter = 0


def recur(val, num):
    global counter

    if val > num:
        return
    elif val == num:
        counter += 1
    
    recur(val+1, num)
    recur(val+2, num)
    recur(val+3, num)


def solution():
    tc = int(sys.stdin.readline())
    global counter 
    for _ in range(tc):
        
        counter = 0
        num = int(sys.stdin.readline())
        recur(0, num)
    
        print(counter)
        

if __name__ == "__main__":
    solution()

```
